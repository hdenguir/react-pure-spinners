Pure CSS Spinners for React

## Instalation

### npm

```sh
npm i react-pure-spinners
```

### yarn

```sh
yarn add react-pure-spinners
```

### unpkg

```html
<script src="https://unpkg.com/react-pure-spinners@latest/dist/bundle.umd.js"></script>
```

## Usage

```js
import { Ellipsis } from 'react-pure-spinners'
```
